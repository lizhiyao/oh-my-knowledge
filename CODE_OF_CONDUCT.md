# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Summary

Be respectful. Assume good intent. Give and receive feedback gracefully. Keep discussion focused on the work.

## Reporting

If you observe behavior that violates the Contributor Covenant, contact the maintainer privately:

**Email:** coderdancestudio@gmail.com

Reports are handled confidentially. This is a single-maintainer project — response is best-effort, typically within a few days.

## Enforcement

The maintainer will follow the [Contributor Covenant Enforcement Guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines) — a graduated response from private correction up to removal from project spaces. Decisions are at the maintainer's discretion; appeals can be sent to the same email.

## Scope

This applies to all project spaces (GitHub issues, PRs, discussions, commits) and to situations where someone is representing the project in public.
